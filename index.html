// INSTALLATION

Up and running in under a minute

Download, run the installer, done. Shortcuts are created automatically.

  1. Download the installer

    Grab PYCORE_IDE_Setup.exe using the button on this page.

  2. Run the setup

    Double-click PYCORE_IDE_Setup.exe and follow the wizard. If Windows shows a blue "Windows protected your PC" box, click More info → Run anyway — the app is safe, just unsigned.

  3. Choose where to install

    Pick an install location and whether you want a desktop shortcut. Tip: to use pip install without admin rights, install to a user folder (e.g. C:\PYCORE IDE) instead of Program Files.

  4. Launch PYCORE IDE

    Open it from your desktop, or just type "PYCORE" in the Windows search bar. You're ready to code.

  5. (Optional) Turn on AI

    Open ⚙ Settings, pick an AI provider, paste your key or PYCORE AI URL, and Save. Now error help works.

  6. (Optional) Install packages

    Type a command in the bottom box, e.g. pip install requests, and hit Run pip. The package stays in the app folder for next time.

// SYSTEM REQUIREMENTS

What you need

Light on requirements — it's designed to run on basic school lab machines.

  • Operating systemWindows 10 / 11 (64-bit)
  • Python pre-installedNot required
  • RAM2 GB minimum, 4 GB recommended
  • Disk space~300 MB (bundled Python + app)
  • InternetOnly for pip & AI features
  • Admin rightsNot needed — portable

On locked-down thin clients, running .exe files may be blocked by policy. Test from a USB drive first; if it's blocked, ask your lab admin.

Optional: PYCORE AI (local)

Want free AI help with no API key? Run a local model on your own PC and point the IDE to it:

# on your PC, with Ollama installed
ollama serve
# expose it with a Cloudflare tunnel
cloudflared tunnel --url http://localhost:11434

Paste the resulting https://…trycloudflare.com URL into Settings → PYCORE AI. Keep your PC and tunnel running while students use it.
