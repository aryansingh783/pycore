# 🔥 Features

## 🤖 AI Error Explanation
Understand Python errors in simple Hinglish.

Example:
```python
x = [1,2,3]
print(x[5])
```

PYCORE explains:
- kya error hai
- kyun aaya
- kaise fix kare
- correct code kya hoga

---

## 🎨 Multiple Themes

Available themes:
- Cyberpunk
- Matrix
- Hacker
- Neon
- Midnight
- Aurora

Use:
```python
:theme matrix
```

---

## ⚡ Smart Commands

| Command | Description |
|---|---|
| `:help` | Show commands |
| `:vars` | Show variables |
| `:stats` | Session stats |
| `:history` | Command history |
| `:explain` | Explain last code |
| `:theme` | Change theme |
| `:clear` | Clear screen |
| `:run file.py` | Run Python file |
